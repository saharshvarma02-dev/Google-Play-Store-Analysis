# Google Play Store Analytics Dashboard

## Project Overview

This project presents a comprehensive analysis of Google Play Store application data with the objective of deriving actionable business insights. It focuses on evaluating app performance, user engagement, monetization strategies, and user sentiment.

The analysis transforms raw data into a structured, interactive HTML dashboard designed for intuitive exploration and decision-making. The project demonstrates end-to-end data analytics capabilities, including data preprocessing, feature engineering, visualization, and reporting.

---

## Dataset Description

The dataset contains detailed information about applications available on the Google Play Store. Key attributes include:

* Application Name
* Category
* Rating
* Number of Reviews
* Number of Installs
* Price
* Application Type
* Last Updated Date
* User Reviews

---

## Data Preprocessing and Transformation

The dataset required extensive cleaning and transformation to ensure analytical accuracy. The following steps were performed:

* Removal of missing, duplicate, and inconsistent records
* Cleaning of special characters such as commas, plus signs, and currency symbols
* Integration of application data with user review data
* Application of sentiment analysis on user reviews

---

## Key Performance Indicators (KPIs)

The dashboard highlights the following core metrics:

* Total number of applications
* Average application rating
* Total installs across all applications
* Total estimated revenue
* Distribution of user sentiment

---

## Key Insights

* Categories with high install volumes indicate strong user demand and engagement.
* Paid applications, although fewer in number, contribute significantly to overall revenue.
* Applications with higher ratings tend to have more positive user sentiment.
* Market growth trends suggest increasing adoption of mobile applications over time.
* Reviews and installs show strong correlation, highlighting the importance of user engagement in driving visibility and success.
